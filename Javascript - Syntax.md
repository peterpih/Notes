<h2>Arrays</h2>
var array = [1,2,3,4];  
<pre>
 [1, 2, 3, 4]
 </pre>
 
array.<b>shift();</b>   
<pre>
&gt; array
(3) [2, 3, 4]
</pre>

array.<b>unshift(9);</b>   
<pre>
4
&gt; array
(4) [9, 2, 3, 4]
</pre>

array.splice(1,2);
<pre>
(2) [2, 3]
</pre>

array.<b>push</b>  - <em>add tot he end of array </em>
