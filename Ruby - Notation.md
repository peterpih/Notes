':something' is a string

'something:' is a symbol

'=>' is a hash operator
<pre>
{ :minimum => 5 }
h = {'dog' => 'canine', 'cat' => 'feline', 'donkey' => 'asinine', 12 => 'dodecine'}  
h = {:dog => 'canine', :cat => 'feline', :donkey => 'asinine', 12 => 'dodecine'}   
h = {dog: 'canine', cat: 'feline', donkey: 'asinine', 12: 'dodecine'}  
</pre>
is a has with 'minimum' as its key and a value of 5.
